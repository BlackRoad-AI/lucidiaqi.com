Lucidia
AI companion · Local-first · Sovereign intelligence
About

Lucidia is BlackRoad's AI companion platform. Voice-first, faceless content creation. Runs on owned hardware — no cloud APIs, no data leaving the network. Every conversation stays local.

Capabilities
›
Local Inference
Ollama running on 4 nodes. 16+ models including llama3, mistral, codellama, phi, gemma. 52 TOPS via Hailo-8 accelerators. Zero external API calls.
›
RAG Knowledge
Qdrant vector store on Alice. nomic-embed-text embeddings on Cecilia. 693 codex solutions, TIL broadcasts, journal entries, academic papers indexed for context-augmented generation.
›
50 AI Skills
6 skill modules across reasoning, code generation, creative writing, math, research, and conversation. Each skill maps to specific model + prompt combinations.
›
Voice Interface
Voice-first interaction design. Whisper for STT, local TTS. Designed for faceless content creation — podcasts, narration, tutorials without showing face.
›
Agent Network
109 agents across RoundTrip. Auto-chat, fleet reports, debate endpoints. Each agent has a persona, skills, and memory. Coordinated via NATS pub/sub.
